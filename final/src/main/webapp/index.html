12201731 박승연 - 방향키 게임

게임 설명

난이도 EASY

방향키가 가리키는 방향이 위, 혹은 아래 두 가지 경우만 주어진다.


난이도 NORMAL

방향키가 가리키는 방향이 상하좌우 네 방향으로 주어진다. 만약 방향키를 잘못 입려한 경우 경고 문구를 띄운다.


난이도 HARD

난이도 NORMAL의 조건을 만족한다.
만약 방향키를 잘못 입력했을 경우 경고 문구를 띄우고, 누적된 틀린 횟수만큼 점수가 감소한다.
점수는 0 보다 낮아질 수 없다.
다음 순서로 입려해야 할 화살표의 방향이 보이지 않는다.
